Form tag

Tag that wraps around all the form elements. "action" attribute points to where the data should be submitted

Attributes

Description of the attribute

Button tag

Every form needs a button to submit.

Label tag

Adds a name to any field. Every field must have a label.

Input types

Adds an input field of varying types. Must always have an id to associate a label. The type="…" attribute defaults to text.

text

Single line text field. If you want, you can leave type="text" off the input.

tel

For telephone numbers. There is no restricted format to accommodate all different countries..

Time-related input

time

For accepting time: hours, minutes, seconds.

week

For accepting a specific week; likely shows a calendar picker.

month

For accepting a specific month; likely shows a calendar picker.

date

For accepting dates; shows a calendar picker.

datetime-local

For accepting a both a date and a time; shows a calendar & time picker.

Number-related input

range

For selecting from a range of number.

number

Accepts numbers; has a up/down switch.

Upload-related input

color

For picking a specific colour; shows a colour palette. Outputs as a hex colour.

url

Accepts a valid website URL.

file

For choosing a file to upload. Use accept="…" to limit filetypes.

search

Specifies the input as a search field.

email

Accepts valid email addresses.

password

For passwords; hides typed characters.

Validation-related input

checkbox

Turns the input into a check toggle.

radio

Turns the input into a radio button. All radio buttons in the group should have the same name="…"
